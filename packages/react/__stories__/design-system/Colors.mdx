## Semantic Tokens

CSS variables (`--*`) that give meaning to the base palette. Defined on
`:root` (light theme) and overridden in `.dark` (dark theme). Exposed to
Tailwind via `@theme inline` as `bg-*` / `text-*` / `border-*`.
