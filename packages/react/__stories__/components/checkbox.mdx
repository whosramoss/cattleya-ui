### states

#### unchecked

`unchecked` is the checkbox's default state.

<Canvas of={CheckboxStories.Unchecked} />

#### checked

The `checked` state shows an animated check mark inside the indicator.

<Canvas of={CheckboxStories.Checked} />

#### indeterminate

The `indeterminate` state is used when the checkbox represents a group of
options that are only partially selected.

<Canvas of={CheckboxStories.Indeterminate} />

### disabled

Checkboxes with the `disabled` prop become inert and communicate that the
option is currently unavailable.
