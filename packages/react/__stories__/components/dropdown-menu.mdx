### basic

`basic` renders the menu content inline, pushing the layout below it. Click
the trigger to toggle it open and closed.

<Canvas of={DropdownMenuStories.Basic} />

### floating

With the `floating` prop, `DropdownMenuContent` is absolutely positioned so
it overlaps the content below it instead of pushing it down.
