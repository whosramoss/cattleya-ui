### type

#### single

With `type="single"`, only one item can be expanded at a time. Pairing it
with `collapsible` allows the open item to be closed again.

<Canvas of={AccordionStories.Single} />

#### multiple

With `type="multiple"`, any number of items can be expanded at the same
time.

<Canvas of={AccordionStories.Multiple} />

### disabled item

Individual items can receive the `disabled` prop, preventing their trigger
from being toggled.

<Canvas of={AccordionStories.DisabledItem} />
